
# Mobile Testing
Here is how to run the project, call api




## System Requirements

- **Java 21+**
- **Visual Studio**
- **Node.js (v14 or higher)**
## Step to run project

### Step 1: Clone project

**Pre-requisite**: Git is required to clone

1. Open windows explorer
2. Choose the directory to clone 
3. Open git bash and copy paste this:

```bash
git clone https://github.com/gAmma05/Date-Time-Validation-By-AI-Testing
```

### Step 3: Open the repository folder through IDE

1. Open Visual Studio Code
2. Open the folder where repository is placed


### Step 4: Setup

1. Setup dependencies

```bash
# Install dependencies
npm init -y

# Install gen AI
npm install @google/generative-ai
```

2. Change type in node modules
Open the node modules (must do step 3 before doing this)
-> Find "type", change "commonjs" into "module"

3. Run the project
```bash
node main.js
```
